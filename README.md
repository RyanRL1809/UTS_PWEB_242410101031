# 1031_UTSPWEB

Saya membuat sebuah aplikasi web simulasi bernama AdoptMe. Web ini adalah platform yang menghubungkan hewan rescue (anjing dan kucing) dengan orang yang ingin mengadopsi. Web ini dilengkapi sistem login sederhana, halaman dashboard, profil pengguna, dan halaman daftar hewan peliharaan.
